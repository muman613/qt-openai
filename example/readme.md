# Examples Directory

This directory contains test applications demonstrating various features of the `qt-openai` library. Each example showcases specific use cases, helping you integrate the library into your Qt applications.

## Contents

| Example Application    | Description                                                                                   |
|-------------------------|-----------------------------------------------------------------------------------------------|
| **test-download-mgr**   | A test application demonstrating how to manage and download resources using Qt's networking capabilities. |
| **test-image-gen**      | A command-line application that interacts with OpenAI's image generation API to create images from text prompts. |
| **test-image-gen-gui**  | A GUI-based application showcasing how to generate images from text prompts using OpenAI's image generation API with a user-friendly interface. |
| **test-tts-gen**        | A command-line application for generating text-to-speech (TTS) audio using OpenAI's API.      |
| **test-tts-gen-gui**    | A GUI-based application for generating TTS audio with OpenAI's API, featuring an interactive user interface. |

## Usage

Each example application is located in its own subdirectory and contains:

- **Source Code**: The implementation of the example.
- **CMakeLists.txt**: A CMake configuration file to build the application.
